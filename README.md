# Prime-Gadget-Zone

A modern and user-friendly e-commerce platform for selling gadgets online. Prime-Gadget-Zone provides a seamless shopping experience with product management, order processing, email notifications, and an intuitive admin dashboard.

## 🚀 Features

### 🛍️ Customer Features

* Responsive and modern UI/UX
* Product browsing and search
* Category-based product listing
* Shopping cart functionality
* Secure checkout process
* Order confirmation email
* User registration and login
* Order history tracking

### ⚙️ Admin Features

* Product management (Add, Edit, Delete)
* Category management
* Order management dashboard
* Customer information management
* Sales and order monitoring

### 📧 Email Notification System

* Automatic email notification to the website owner when a new order is placed
* Order confirmation email sent to customers
* Detailed order information included in emails

### 🔒 Security

* Secure authentication system
* Protected customer data
* Input validation and error handling

## 🛠️ Technologies Used

* Frontend: HTML, CSS, JavaScript, React.js
* Backend: Node.js, Express.js
* Database: MongoDB
* Authentication: JWT
* Email Service: Nodemailer
* Hosting: Vercel / Netlify / Render

## 📧 Email Workflow

1. Customer places an order.
2. Order data is stored in the database.
3. Website owner receives a notification email.
4. Customer receives an order confirmation email.
5. Admin can manage the order from the dashboard.

## 🎯 Project Goal

The goal of Prime-Gadget-Zone is to provide a professional online gadget marketplace with a smooth shopping experience, automated order handling, and efficient communication between customers and administrators.
